# Compostavel App

## 📝Description

<h1 align="center"></h1>

    
<p align="center">🚀 Application aims to help people in the creation and maintenance of composters, and it also has support for donors to donate inputs to the composters. .</p>

<h1 align="center"></h1>

## 🗺️Features
✅ Registration and login
❌ Composter CRUD
